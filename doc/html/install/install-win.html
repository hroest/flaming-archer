OpenMS Installation on Windows

Warning: Installation of OpenMS on Windows is highly experimental!

Setup of environment

  1. Install MinGW (5.1.3)
    • Install candidate version (not the current release because of differing gcc versions - 3.4.2 vs. 3.4.5)
    • Install to default location (c:/MinGW/)
    • Select gcc/g++ only (no make)
  2. Install MSYS (1.0.10)
    • Install to default location (c:/msys/)
    • Ignore windows API warning
  3. Install msysDTK (1.0.1)
  4. Install MSYS autoconf 2.59
  5. Install Qt (4.3.1)

Contrib installation

  1. Copy the contrib package to OpenMS/contrib
  2. Execute autoconf
  3. Execute ./configure
  4. Execute make

OpenMS installation

  1. Add Qt bin folder to PATH (this is where Windows looks for the dll)
  2. Execute autoconf
  3. Execute ./configure
    • Explicitly state where Qt lib, include, moc and uic are (/c/Qt/3.4.1/...)
  4. Execute make
  5. ...
