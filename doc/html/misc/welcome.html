Installation instructions
On these pages you find the OpenMS installation instructions for Linux and MacOS.
OpenMS + TOPP documentation
There are three ways to obtain the OpenMS/TOPP documentation.
  • The documentation of the latest release is available online
  • The documentation of the latest release can be downloaded of the SourceForge files section files section
  • The documentation can be build from this source package (e.g. for development versions).

If your OpenMS version is a development version from SVN, the following applications are needed in order to create the OpenMS documentation:

  • doxygen (version 1.4.5 or higher is recommended as it is is aware of most STL classes)
  • pdflatex (for the pdf version of the tutorial)
  • OpenMS (for the parameter documentation)

The commands used to build the documentation in OpenMS/doc/ are:

  • make doc (for the OpenMS/TOPP documentation including HTML tutorials)
  • make OpenMS_tutorial (for the PDF version of the tutorial)
  • make TOPP_tutorial (for the PDF version of the tutorial)
