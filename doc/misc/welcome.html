Welcome

Here you can find the OpenMS documentation. The documentation constists of the following fixed parts:
  • Home
    This page.
  • Installation
    Detailed installation instructions.
  • Tutorial
    A short tutorial for programmers.
  • FAQ
    You should give this page a try if you have a problem using OpenMS.
Additionally the following parts can be built:
  • Coding conventions
    A pdf document describing the coding style of OpenMS.
    Note: Execute 'make style' in the OpenMS/source directory to create it.
  • Class documentation
    The doxygen documentaion of the OpenMS classes.
    Note: Execute 'make doc' in the OpenMS/source directory to create it. Doxygen version 1.4.5 or higher is recommended as it is is aware of most STL classes.
